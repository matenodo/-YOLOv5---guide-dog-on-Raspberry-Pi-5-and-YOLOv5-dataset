# 基于 YOLOv5s 的智能导盲系统

本项目设计并实现了一套基于 YOLOv5s 目标检测算法和树莓派 5B 嵌入式平台的智能导盲系统。系统通过摄像头实时采集环境图像，利用轻量化部署的 ONNX 模型对行人、车辆、坑洞、楼梯、红绿灯等 11 类障碍物进行检测与识别，并通过多优先级语音模块进行实时播报，辅助视障人士安全出行。

## 📖 项目背景

我国有超过 1700 万视力残疾人，而导盲犬数量极少、培养成本高昂，现有电子导盲设备存在感知单一、缺乏语义理解或成本过高等问题。本项目旨在计算机视觉方案中寻找性能与成本的平衡点，为视障人士提供一种高效、低成本的辅助出行方案。

## 🎯 主要功能

- 实时检测 11 类目标：坑洞、障碍物、路障、行人、车辆、斑马线、树木、杆子、台阶、红灯、绿灯
- 输出目标方位（左前方、前方、右前方）及类别信息
- 多优先级语音告警（高危目标立即播报，普通目标根据距离和数量过滤）
- 支持冷却机制，避免重复播报
- 基于树莓派 5B + CSI 摄像头 + USB 扬声器，完全离线运行

## 🖥️ 系统架构

### 硬件组成：

- **主控**：树莓派 5B（8GB 内存）
- **视觉模块**：Raspberry Pi Camera Module 1（CSI 接口）
- **语音输出**：USB 声卡 + 扬声器（JLAC6082A2）
- **电源**：6200mAh 可充电锂电池组

### 软件框架：

- 操作系统：Raspberry Pi OS（64 位）
- 运行环境：Python 3.9 + PyTorch 2.0（ARM64）+ OpenCV + ONNX Runtime
- 检测模型：YOLOv5s 迁移学习训练 → 转换为 ONNX 格式（输入尺寸 416×416）
- 语音引擎：pygame.mixer（预录 OGG 音频文件）
